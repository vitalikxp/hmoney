# hmoney

Учёт личных финансов. Pet-project — упрощённый аналог HoneyMoney.

**Домен**: https://money.vitalik.dev

## Требования

- Node.js 20+

## Установка и запуск

```bash
npm ci
npm run dev
```

Открыть http://localhost:5173

## Скрипты

| Команда | Описание |
|---------|----------|
| `npm run dev` | Dev-сервер с HMR |
| `npm run build` | TypeScript check + production сборка в `dist/` |
| `npm run preview` | Локальный preview собранного `dist/` |

## Технологии

React, TypeScript, Vite, Tailwind CSS, Firebase (в плане), PWA (в плане).
