# Исследование: Сайт HoneyMoney (hmbee.ru) + демо-приложение

**Источник**: https://hmbee.ru + https://demo.hmbee.ru
**Дата обращения**: 2026-05-15

## Ключевые выводы

HoneyMoney — это приложение для ручного учёта личных финансов.

## Анализ демо-приложения

Демо-версия полностью функциональна (сброс каждые 10 мин).

### Навигация

Четыре основных раздела:

`Счета и Конверты → Календарь → Список → Отчёты`

Переключение горячими клавишами: Alt+1, Alt+2, Alt+3, Alt+4.

### 1. Календарь (основной экран)

- Сетка: недели (строки) × дни (колонки)
- Каждый день: список транзакций (сумма, категория курсивом, описание)
- Сверху: переключение месяцев (« апр / Май 2026 / июнь »), кнопка «Статистика», кнопка «Новая транзакция»
- Сводка за месяц: доходы +$6 626 (62% не потрачено), расходы –$2 538, разница +$4 088
- Net Worth в хедере (ALT+X): ХаниМани $10 947 + Резервы $3 103 + Фонды $1 111 + Цели $7 777 = $22 938
- Переключение вида: «кратко / подробно»

### 2. Форма создания транзакции (модал)

- **Тип**: расход (по умолчанию), переключается на доход
- **Режим**: Факт / План
- **Дата**: Datepicker «Май 15, 2026»
- **Сумма + описание**: Natural language input
  - Формат: `5*250 яблоки` = 5 шт × 250 = 1250
  - Текст в скобках игнорируется: `(комментарий)`
  - Плейсхолдер: `5*250 яблоки (100 в скобках не учтётся)`
- **Валюта**: селектор (USD, RUB, EUR, CAD...)
- **Счёт**: комбобокс со всеми счетами и текущим балансом
- **Конверт**: комбобокс с конвертами
- **Категория**: в HM — autocomplete с иерархией через `/`; в нашем проекте — простое текстовое поле (строка без справочника)
- **Кнопка**: «Внести расход»

**Горячие клавиши формы**: Ctrl+Enter = сохранить, Esc = закрыть

### 3. Статистика (ALT+Z)

Панель с двумя колонками:

**Расходы по категориям** (–$2 538):
- food –$685, baby –$492, house –$396, insurance –$242, car –$241, health –$217, fun –$175
- При нажатии на группу — раскрытие подкатегорий
- Кнопка «Подкатегории» для переключения вида

**Доходы по категориям** (+$6 626):
- corporation +$6 118 (92%), cashback +$343 (5%), freelance +$165 (2%)
- Процент от общего дохода

> В нашем проекте категория — плоская строка, без подкатегорий и иерархии.

### 4. Счета и Конверты

**Счета** — таблица с группировкой:
- Favorites: избранные счета
- Investments: инвестиционные с галкой «Исключить из баланса» / «Счёт не участвует в бюджете»
- Hidden: скрытые

Каждый счёт: название, баланс, для кредиток — доступный лимит

**Конверты** — 4 типа:
- ХаниМани (свободные)
- Резервы (Reserves)
- Фонды (Funds) — с колонками «Фонд» и «Баланс»
- Цели (Goals) — с колонками «Цель» и «Баланс»
- Скрытые Фонды / Скрытые Цели

**Действия**: кнопка «Создать», «Новый конверт», «Импортировать из CSV»

### 5. Список транзакций

Таблица с фильтрами:
- По счёту (комбобокс)
- По конверту (комбобокс)
- По типу (доходы и расходы / доходы / расходы / переводы / неподтверждённые)
- По категории (в HM — autocomplete; в нашем проекте — текстовый поиск по строке)
- По описанию (текстовый поиск ИЛИ)

### 6. Отчёты

- **По месяцам** — сравнение периодов
- **Всего денег** — Net Worth (все счета и конверты)

### 7. Редактор категорий (модал из футера)

В HM есть редактор со списком всех категорий, счётчиком и иерархией:

```
freelance (2989)
food / groceries-household (940)
food / eat out (470)
...
```

> В нашем проекте **не будет** редактора категорий — категория это просто строка в транзакции, без справочника, счётчика или иерархии.

### 8. Клавиатурные сочетания (модал из футера)

| Сочетание | Действие |
|-----------|----------|
| Ctrl + Enter | Сохранить открытую транзакцию |
| Esc | Закрыть открытую транзакцию |
| Зажать Alt/Ctrl при перетаскивании в Календаре | Копировать транзакцию |
| Alt + 1 | Перейти в «Счета и Конверты» |
| Alt + 2 | Перейти в «Календарь» |
| Alt + 3 | Перейти в «Список» |
| Alt + 4 | Перейти в «Отчёты» |
| Alt + ← | Предыдущий месяц в Календаре |
| Alt + → | Следующий месяц в Календаре |
| Alt + ↑ | Текущий месяц в Календаре |
| Alt + Z | Показать статистику по категориям |
| Alt + X | Показать прогноз Резервов |

### 9. Курсы валют (модал из футера)

- 172+ валюты, включая криптовалюты (BTC, ETH)
- Курсы к USD
- Ежедневное обновление
- Ссылки на курсы национальных банков (Россия, Украина, Казахстан, Беларусь)

> Мультивалютность — post-MVP.

## Влияние на требования

- Natural language input для суммы и описания — ключевая фича
- Факт/План режимы для транзакций
- Горячие клавиши — неотъемлемая часть UX
- Категория — строка в транзакции, без отдельного справочника
- 4 типа конвертов, а не просто счета
- Типы транзакций: доход, расход, перевод; неподтверждённые — статус
- Фильтры — важная часть UI
- Net Worth — ключевой индикатор на каждом экране
