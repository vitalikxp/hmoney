# UI компоненты

## Маршруты

| Путь | Компонент | Описание |
|------|-----------|----------|
| `/login` | `LoginPage` | Вход/регистрация |
| `/` | `CalendarPage` | Календарь с транзакциями |
| `/transactions` | `TransactionList` | Список всех транзакций |
| `/transactions/new` | `TransactionForm` | Создание транзакции |
| `/transactions/:id/edit` | `TransactionForm` | Редактирование транзакции |
| `/categories` | `CategoryList` | Управление категориями |
| `/accounts` | `AccountList` | Управление счетами |
| `/reports` | `ReportsPage` | Аналитика и отчёты |

## Компоненты (иерархия)

```
App
├── AuthGuard                # защита маршрутов
│   └── Layout
│       ├── Sidebar         # навигация
│       ├── Header          # баланс, месяц, пользователь
│       └── Main
│           ├── CalendarPage
│           │   ├── CalendarGrid        # сетка календаря
│           │   ├── CalendarDay         # день с транзакциями
│           │   └── TransactionPopup    # popup с деталями дня
│           ├── TransactionList
│           │   ├── TransactionFilters  # фильтры
│           │   └── TransactionRow      # строка транзакции
│           ├── TransactionForm
│           │   ├── AmountInput         # ввод суммы
│           │   ├── CategorySelect      # выбор категории
│           │   ├── AccountSelect       # выбор счёта
│           │   └── DatePicker          # выбор даты
│           ├── CategoryList
│           │   ├── CategoryTree        # дерево категорий
│           │   └── CategoryForm        # форма категории
│           ├── AccountList
│           │   ├── AccountCard         # карточка счёта
│           │   └── AccountForm         # форма счёта
│           └── ReportsPage
│               ├── PeriodSelector      # выбор периода
│               ├── PieChart            # круговая диаграмма
│               └── BalanceChart        # график баланса
```
