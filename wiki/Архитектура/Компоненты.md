# UI компоненты

## Маршруты

| Путь | Компонент | Описание |
|------|-----------|----------|
| `/login` | `LoginPage` | Вход/регистрация |
| `/` | `CalendarPage` | Календарь с транзакциями |
| `/transactions` | `TransactionList` | Список всех транзакций |
| `/transactions/new` | `TransactionForm` | Создание транзакции |
| `/transactions/:id/edit` | `TransactionForm` | Редактирование транзакции |
| `/accounts` | `AccountsPage` | Счета и Конверты |
| `/reports` | `ReportsPage` | Отчёты |
| `/reports/net-worth` | `NetWorthPage` | Net Worth |

## Модальные окна (не маршруты, открываются из футера)

| Модал | Триггер | Содержимое |
|-------|---------|------------|
| `KeyboardShortcutsModal` | Ссылка «Клавиатурные сочетания» | Таблица сочетаний |
| `CurrencyRatesModal` | Ссылка «Курсы валют» | 172+ валюты, курсы к USD |
| `ReserveForecastPopover` | ALT+X или клик в хедере | Прогноз Резервов: дефицит/профицит |
| `NewTransactionModal` | Кнопка «Новая транзакция» | Форма транзакции |
| `StatisticsPanel` | ALT+Z или кнопка «Статистика» | Расходы/доходы за период |

## Компоненты (иерархия)

```
App
├── AuthGuard                        # защита маршрутов
│   └── Layout
│       ├── Header
│       │   ├── MonthNav            # переключение месяцев (calendar only)
│       │   ├── ReserveForecastPopover     # ALT+X — прогноз Резервов
│       │   └── UserMenu
│       ├── Navbar                  # Alt+1..4 навигация
│       └── Main
│           ├── CalendarPage
│           │   ├── CalendarStats         # доходы/расходы/разница за месяц
│           │   ├── CalendarGrid          # сетка календаря (недели × дни)
│           │   ├── CalendarDay           # день с транзакциями
│           │   ├── TransactionPopup      # popup с деталями дня
│           │   └── StatisticsPanel       # ALT+Z — статистика
│           ├── TransactionList
│           │   ├── TransactionFilters    # счёт, конверт, тип, категория (строка), поиск
│           │   └── TransactionRow        # строка транзакции
│           ├── TransactionForm           # (модал)
│           │   ├── TypeToggle            # расход/доход + Факт/План
│           │   ├── NaturalLanguageInput  # 5*250 яблоки (коммент)
│           │   ├── CurrencySelect        # выбор валюты
│           │   ├── CategoryInput         # текстовое поле (строка, не справочник)
│           │   ├── AccountSelect         # выбор счёта
│           │   ├── EnvelopeSelect        # выбор конверта
│           │   └── DatePicker            # выбор даты
│           ├── AccountsPage
│           │   ├── AccountList
│           │   │   ├── AccountGroup      # ✅ Favorites, Investments, Hidden
│           │   │   ├── AccountCard       # ✅ карточка счёта с балансом
│           │   │   └── AccountModal      # ✅ форма счёта (модал)
│           │   │       └── IconPicker    # ✅ выбор emoji (18 preset)
│           │   ├── EnvelopeList
│           │   │   ├── EnvelopeGroup     # ХаниМани, Резервы, Фонды, Цели
│           │   │   ├── EnvelopeCard      # карточка конверта
│           │   │   └── EnvelopeForm      # форма конверта
│           │   └── CsvImport             # импорт из CSV
│           └── ReportsPage
│               ├── PeriodSelector        # выбор периода
│               ├── PeriodComparison      # сравнение периодов
│               ├── PieChart              # круговая диаграмма
│               └── BalanceChart          # график баланса
├── NewTransactionModal            # кнопка «Новая транзакция»
├── KeyboardShortcutsModal         # справка по горячим клавишам (футер)
└── CurrencyRatesModal             # 172+ валюты с курсами к USD (футер)
```

## Статус реализации

Компоненты, отмеченные ✅, реализованы. Остальные — в плане.
