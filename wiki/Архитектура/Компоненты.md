# UI компоненты

## Маршруты

| Путь | Компонент | Описание | Статус |
|------|-----------|----------|--------|
| `/login` | `LoginPage` | Вход | ✅ |
| `/register` | `RegisterPage` | Регистрация | ✅ |
| `/accounts` | `AccountsPage` | Счета | ✅ |
| `/envelopes` | `EnvelopesPage` | Конверты | ✅ |
| `/*` | redirect → `/accounts` | — | ✅ |
| `/` | `CalendarPage` | Календарь с транзакциями | план |
| `/transactions` | `TransactionList` | Список всех транзакций | план |
| `/reports` | `ReportsPage` | Отчёты | план |

## Модальные окна (не маршруты, открываются из футера)

| Модал | Триггер | Содержимое |
|-------|---------|------------|
| `KeyboardShortcutsModal` | Ссылка «Клавиатурные сочетания» | Таблица сочетаний |
| `CurrencyRatesModal` | Ссылка «Курсы валют» | 172+ валюты, курсы к USD |
| `ReserveForecastPopover` | ALT+X или клик в хедере | Прогноз Резервов: дефицит/профицит |
| `NewTransactionModal` | Кнопка «Новая транзакция» | Форма транзакции |
| `StatisticsPanel` | ALT+Z или кнопка «Статистика» | Расходы/доходы за период |

## Компоненты (иерархия)

```
App (BrowserRouter)
├── /login      → LoginPage    (public)
├── /register   → RegisterPage (public)
├── /accounts   → AuthGuard → AccountsPage  ✅
├── /envelopes  → AuthGuard → EnvelopesPage ✅
└── *           → redirect /accounts
```

> **Реальная структура Layout**: `Layout` — компонент внутри каждой страницы (`AccountsPage → Layout`, `EnvelopesPage → Layout`), не обёртка вокруг AuthGuard. Layout содержит: header (логотип + desktop nav + logout) + main + mobile bottom nav. Заголовочные под-компоненты (MonthNav, ReserveForecastPopover, UserMenu) — план, ещё не реализованы.

