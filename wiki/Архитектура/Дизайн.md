# Дизайн-система: ClickHouse

Выбранная дизайн-система для hmoney — **ClickHouse Design System** (yellow-on-black, data-dense).

## Обоснование выбора

| Критерий | Оценка |
|---|---|
| Палитра «пчелиная» | yellow-on-black (`#faff69` + `#0a0a0a`) |
| Плотность данных | Data-dense dashboards — календарь транзакций |
| Fintech-контекст | Техническая точность, подходит для финансового учёта |
| Темы | Dark (primary) + Light (системная `prefers-color-scheme`) |
| Кастомизация | Single-accent system — легко заменить `#faff69` на свою валюту |

Альтернативы: Binance (fintech + желтый), Kraken (data-dense + purple), Revolut (digital banking).

## Палитра

### Brand
- **Primary (Electric Yellow)**: `#faff69` — все CTA, акценты, статистика
- **Primary Active**: `#e6eb52` — hover
- **Primary Disabled**: `#3a3a1f` — disabled state

### Surface
- **Canvas**: `#0a0a0a` — фон страницы
- **Surface Soft**: `#121212` — разделители секций
- **Surface Card**: `#1a1a1a` — карточки, код, цены
- **Surface Elevated**: `#242424` — вложенные карточки
- **Hairline**: `#2a2a2a` — границы 1px

### Text
- **Ink (on-dark)**: `#ffffff` — заголовки
- **Body**: `#cccccc` — основной текст
- **Muted**: `#888888` — подписи, футер
- **Muted Soft**: `#5a5a5a` — третичный текст
- **On Primary (on-yellow)**: `#0a0a0a` — текст на жёлтом

### Semantic
- **Emerald**: `#22c55e` — успех
- **Rose**: `#ef4444` — ошибка
- **Blue**: `#3b82f6` — информация
- **Warning**: `#f59e0b` (`#d97706`) — предупреждение

## Светлая тема (Light Mode)

ClickHouse — dark-only, поэтому светлая тема сделана вручную инвертированием поверхностей и текста. Жёлтый акцент `#faff69` сохранён без изменений.

Переключение: автоматически через `@media (prefers-color-scheme: light)`. Тёмная тема — первичная (default), светлая — вторичная (по системной настройке).

**Адаптация ссылок**: желтый `#faff69` не контрастирует на белом фоне, поэтому в светлой теме ссылки (`text-link`) используют акцентный синий `#2563eb`. В тёмной теме ссылки остаются жёлтыми `#faff69`.

## Типографика

- **Гарнитура**: Inter (всё, кроме кода)
- **Моноширина**: JetBrains Mono (код, числа)
- **Display**: 700 weight, негативный tracking (-1 to -2.5px)
- **Body**: 400 weight
- **Buttons/Nav**: 600 weight, 14px

| Токен | Размер | Weight | Использование |
|---|---|---|---|
| display-xl | 72px | 700 | Hero h1 |
| display-lg | 56px | 700 | Секции |
| display-md | 40px | 700 | Подсекции |
| display-sm | 32px | 700 | Заголовки карточек |
| title-lg | 24px | 700 | Названия планов |
| title-md | 18px | 600 | Заголовки карточек |
| body-md | 16px | 400 | Основной текст |
| body-sm | 14px | 400 | Мелкий текст |
| caption | 13px | 500 | Бейджи |
| code | 14px | 400 | JetBrains Mono |

## Радиусы

| Токен | Значение | Применение |
|---|---|---|
| xs | 4px | Бейджи |
| sm | 6px | Маленькие кнопки |
| md | 8px | CTA-кнопки, инпуты |
| lg | 12px | Карточки контента |
| pill | 9999px | Бейджи-пилюли |

## Отступы (spacing)

4px base unit: 4 / 8 / 12 / 16 / 24 / 32 / 48 / 96px

## Компоненты (ключевые для hmoney)

- **button-primary**: `#faff69` bg + `#0a0a0a` text, 8px radius, 40px высота
- **button-secondary**: `#1a1a1a` bg + `#ffffff` text
- **text-input**: `#1a1a1a` bg, focus border `#faff69`
- **category-tab** / **category-tab-active**: табы навигации
- **badge-pill** / **badge-yellow**: пилюли для статусов
- **feature-card-dark**: карточки для контента (12px radius, 32px padding)

## Вопросы и противоречия

> Вопросы ниже закрыты с принятыми решениями. Оставлены для истории.

- ✅ **Yellow single-accent**: дополнен semantic цветами (emerald для доходов, rose для расходов).
- ✅ **Inter и JetBrains Mono** — open-source, хорошо ложатся на Tailwind стек.
- ✅ **Light mode**: сделан вручную. Принято. При необходимости — дизайн-система с нативной поддержкой обеих тем (Binance).
